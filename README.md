# atmotool
Netatmo web client

Deployment & Running
--------------------

### Node.js

    brew install nodejs
    brew install npm

    npm install bower -g

    npm install
    bower install

### Open in a browser

[http://localhost:3000/index.html](http://localhost:3000/index.html)
